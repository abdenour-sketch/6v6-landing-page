# Privacy Policy and Terms & Conditions Landing Page

A responsive landing page for displaying Privacy Policy and Terms & Conditions.

## Features
- Clean, modern design
- Mobile-responsive layout
- Easy to customize
- Ready for GitHub Pages

## Setup
1. Clone this repository
2. Replace the placeholder content in `index.html` with your own
3. Add your `privacy-policy.pdf` file
4. Commit and push to GitHub
5. Enable GitHub Pages in your repository settings

## Customization
- Update the contact email in `index.html`
- Replace the placeholder company name in the footer
- Add your own `privacy-policy.pdf` file
